Handlers which relate to the player object controller.

Handlers - qtEnableStepButtons, qtEnableVolumeControl, qtGetMovieControllerInfo(), qtGetMovieControllerType(), qtServiceController

command qtEnableStepButtons MovieControllerID, Enable
Description:
Enables/disables the step forward/back buttons in the controller.
Parameters:
MovieControllerID: movieControllerID of movie to rotate.
Enable: True/false.
Return:
empty
command qtEnableVolumeControl MovieControllerID, Enable
Description:
Turns volume controls on and off for a controller.
Parameters:
MovieControllerID: movieControllerID of player of overlay movie on.
Enable: True/false.
Return:
empty
function qtGetMovieControllerInfo (MovieControllerID, Property)
Description:
Returns enabled information about the controller.
Parameters:
MovieControllerID: movieControllerID of movie.
Property: Property to get enabled state for (undo,cut,copy,paste,clear,selectall,selectnone,isplaying)
Return:
true/false
function qtGetMovieControllerType (MovieControllerID)
Description:
Returns the type of controller for a moviecontroller as stored in the movies UserData properties. This can be set in program like LiveStage Pro. If no value is stored in the file then empty ("") is returned. "panorama_10" and "object_10" represent version 1 panorama and object controls.
Parameters:
MovieControllerID: movieControllerID of movie.
Return:
qtvr, panorama, panorma_10, object, object_10, none, standard, empty("")
handler qtServiceController (MovieControllerID)
Description:
Manually services a movie in a movie controller (MoviesTask).
Parameters:
MovieControllerID: movieControllerID of movie to rotate.
Return:
empty.
