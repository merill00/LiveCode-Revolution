Handlers for interacting with Flash tracks in a player object.

Handlers - qtClickFlashButton, qtGetFlashVariable(), qtGoToFlashFrame, qtGoToFlashLabel, qtSetFlashVariable

command qtClickFlashButton MovieControllerID, TargetTrackIndex, ButtonPath, ButtonID, ButtonStateTransition
Description:
Clicks a button in a flash track.
Parameters:
MovieControllerID: ID of the Revolution player.
TargetTrackIndex: Target track index.
ButtonPath: Path to button.
ButtonID: ID of button to click.
ButtonStateTransition: Button state transition. Possible values are OverUpToIdle, OverUpToOverDown, OverDownToOverUp, OverDownToOutDown, OutDownToOverDown, OutDownToIdle, IdleToOverDown, OverDownToIdle, IdleToOverUp.
Return:
empty or string if error
function qtGetFlashVariable (MovieControllerID, TargetTrackIndex, VarPath, VarName)
Description:
Sets a variable in a flash track.
Parameters:
MovieControllerID: ID of the Revolution player.
TargetTrackIndex: Target track index.
VarPath: Path to flash variable.
VarName: Name of variable.
Return:
empty or string if error
command qtGoToFlashFrame (MovieControllerID, TargetTrackIndex, FrameNumber)
Description:
Navigates to a frame in a flash track.
Parameters:
MovieControllerID: ID of the Revolution player.
TargetTrackIndex: Target track index.
FrameNumber: Frame number.
Return:
empty or string if error
command qtGoToFlashLabel (MovieControllerID, TargetTrackIndex, FrameLabel)
Description:
Navigates to a label in a flash track.
Parameters:
MovieControllerID: ID of the Revolution player.
TargetTrackIndex: Target track index.
FrameLabel: Frame label.
Return:
empty or string if error
command qtSetFlashVariable (MovieControllerID, TargetTrackIndex, VarPath, VarName, VarValue)
Description:
Sets a variable in a flash track.
Parameters:
MovieControllerID: ID of the Revolution player.
TargetTrackIndex: Target track index.
VarPath: Path to flash variable.
VarName: Name of variable to set.
VarValue: Value to assign to variable.
Return:
empty or string if error
